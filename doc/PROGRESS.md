## Phase 2: 地图构建 ✅ 代码完成

用户遥控小车到各点，命名保存，构建路线地图。

### API

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/map/build/start` | 开始构建，记录起点 |
| POST | `/api/map/control` | 遥控驾驶 `{vx,vy,vz}` |
| POST | `/api/map/waypoint {name}` | 保存当前点为航点 |
| POST | `/api/map/build/end {name}` | 结束构建，保存地图 |
| GET | `/api/map/info` | 当前构建中的信息 |
| GET | `/api/map/list` | 列出所有已保存地图 |

### 数据存储

每条路线保存为独立 JSON 文件：
```
~/Rosmaster-App/rosmaster/maps/<路线名>.json
```

```json
{
  "name": "测试路线一",
  "origin": {"x": 1.23, "y": 4.56, "name": "起点"},
  "waypoints": [
    {"x": 2.34, "y": 5.67, "name": "A"},
    {"x": 3.45, "y": 6.78, "name": "B"}
  ]
}
```

### 导航逻辑
- 从**当前位置**出发（不要求停在某个航点上）
- TSP 暴力枚举计算最短遍历顺序（4-5个点毫秒级）
- 全程激光避障，遇障绕行
- 说"停"立即停止
